# 🌸 Iris Dataset Visualization Project

## 📌 Overview

This project focuses on visualizing the Iris dataset to understand the differences between flower species. The dataset contains measurements of iris flowers, and visualization helps to identify patterns and relationships between features.

## ⚙️ How It Works

The Iris dataset is loaded and checked for missing values. Important features such as sepal length, sepal width, petal length, and petal width are selected.

Different types of plots are created to explore the data:

* Scatter plots to compare features
* Pair plots to see relationships between all features
* Histograms to understand data distribution

These visualizations help in understanding how the three flower species (Setosa, Versicolor, Virginica) differ from each other.

## 🧠 Techniques Used

* Data Visualization
* Data Exploration

## ▶️ How to Use

1. Install required libraries: pandas, matplotlib, seaborn
2. Run the notebook or Python file
3. View the generated plots

## 📊 Example

* Scatter plot showing relationship between petal length and petal width
* Pair plot showing all feature relationships
* Histogram showing distribution of sepal length

## 🛠️ Tools

Python, Pandas, Matplotlib, Seaborn

## ⚠️ Limitations

* No prediction or classification is performed
* Only useful for understanding data visually

## ✅ Conclusion

This project helps in understanding the Iris dataset through visualization, making it easier to see patterns and differences between flower species.
